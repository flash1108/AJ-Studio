# AJ Studio Support & Privacy Center

This project is a static website for AJ Studio, serving as a support and privacy center for various mobile applications.

## Project Overview
- **Goal**: Provide support links, contact information, and privacy policies for AJ Studio apps.
- **Tech Stack**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla).
- **Structure**:
  - `index.html`: Main landing page with app listings and contact form links.
  - `app-ads.txt`: AdMob authorization file.
  - `privacy.html`: (Pending) Privacy policy for "FairShare - Easy Bill Splitter".

## Development Guidelines
- **Visuals**: Maintain a clean, modern aesthetic with a focus on readability and accessibility.
- **Styling**: Use Vanilla CSS. Avoid external CSS frameworks unless requested.
- **Localization**: Support both Traditional Chinese (zh-Hant) and English (en). Use the existing class-based language switching logic in `index.html`.
- **Responsive Design**: Ensure the site is mobile-friendly.
